# @openpalette/core

A library for interacting with OpenPalette data.

```bash
npm install --save @openpalette/core
```

OR

```bash
yarn add @openpalette/core
```

## API

- [hexToRgba](#hexToRgba)
- [hexToHsva](#hexToHsva)

---

### `getPalettes`

Returns an array of all palettes.

**Type**: `function getPalettes(): OpenPalette[]`
